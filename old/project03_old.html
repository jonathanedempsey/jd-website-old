VALKYRIE

HTML5 Canvas Game

WHAT IT DOES

A HTML5/JavaScript canvas game was created that allowed users to control Valkyrie, a flying turtle, with the aim of avoiding oncoming missiles. The turtle character is located at the far left of the canvas and is moved up and down by using the W and S keys. The oncoming missiles move from the right of the screen to the left, towards the turtle. The missiles spawn randomly along the Y axis on the right as they leave the canvas to the left. If a missile hits the turtle the missile changes from a missile image to an explosion image, an explosion sound file is played and the turtles Hit Points go down. Once the turtle’s HP hits 0, the game resets.

HOW IT WORKS

The HTML is primarily used to employ the games canvas and to display the turtle’s HP. Multiple canvases are used in the game, one for each asset. The JavaScript is where all the games functionality occur. The audio, images, key bindings and variables are all initialized before the 3 main functions execute. The first function controls the turtle’s HP, the turtle’s movement through the use of key events and the function that runs through the sprite sheet to give the turtle animation. The next function manages the looping background image of the clouds and initiates the game’s background music. The final few functions are used to set the missiles varying speeds, their X and Y values, and the functionality revolved around the collision detection.

WHAT I LEARNED

In this project, I learned how to implement HTML5 canvases which were given functionality through JavaScript. Multiple canvases were implemented in the game. Each canvas is used to layer the one of the games elements, such as the background, turtle and enemies. The turtle’s animation is made by making a function that reads iteratively through a sprite sheet and prints the frames of the sprite sheet into the canvas. Event listeners were made that were bound to the W and S keys so when pressed, they move the turtle sprite up and down the canvas. Collision detection was used between the turtle sprite and enemies to play sound clips, change sprites and affect the players health. Finally, A looping background using two images was used to give the illusion that the turtle is flying through the air.
