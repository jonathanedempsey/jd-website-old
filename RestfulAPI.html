A single page HTML/CSS/JavaScript web application using the JavaScript MVC (Model, View Controller) framework, Backbone JS.

The application dynamically views, adds, deletes and updates the page content, interacts with a back-end web server and handles events within the web page. A RESTful API is created using Node.js, Express and MongoDB. Node.js is a JavaScript run time which uses an event-driven and non-blocking I/O model, which is useful building scalable network applications. Express is a Node.js web application framework which will allow our web application to dynamically render the HTML page based on arguments passed to templates. MongoDB is a simple to use database that uses a JSON data model which maps to applications, and has dynamic schemas that allow quick iterations. For a full overview of the project, refer to the downloadble report.
